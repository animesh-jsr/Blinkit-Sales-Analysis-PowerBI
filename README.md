# Blinkit Sales Analysis Dashboard

## Description
This project presents an interactive Power BI dashboard built using Blinkit sales data. The dashboard provides insights into sales performance, customer ratings, product categories, and outlet characteristics. It helps identify key business trends and supports data-driven decision-making through dynamic visualizations and KPIs.

## Features
- Total Sales Analysis
- Average Sales & Ratings
- Product Category Performance
- Outlet Size and Location Analysis
- Interactive Filters and Slicers
- DAX-Based KPI Calculations

## Tools & Technologies
- Power BI
- DAX
- Power Query
- Excel
